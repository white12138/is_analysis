# 修改密码用例
 **1. 用例规约**

|用例名称|修改密码|
|-------|:-------------|
|功能|修改用户的密码|
|参与者|学生，老师|
|前置条件|必须先登录|
|后置条件|跳转登陆页面|
|主事件流| 1.用户填写新密码（两次输入） <br/> 2.用户提交修改信息 <br/>3.系统存储修改后密码|
|备选事件流| 1.系统提示两次输入的密码不相同 |
