# 接口：login  
- 功能：
    登录

- API请求地址： 
    接口基本地址/v1/api/login

- 请求方式 ：
    POST

- 请求实例：

        {
            "id":"201710414227",
            "password":"894984",
            "type":"学生"
        }
        
- 请求参数说明:        

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |student_id/teacher_id|学生学号或者老师的工号，由type决定。|
  |password|用户的密码| 
  |type|用户类型|
  
- 返回实例：

        { 
            "status": true,
            "info": null,    
        }
 
- 返回参数说明：    
 
  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |status|true表示正确的返回，false表示有错误|
  |info|返回结果说明信息|
