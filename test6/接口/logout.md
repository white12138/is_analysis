# 接口：logout 
- 功能：
  登出
    
- API请求地址： 
    接口基本地址/v1/api/logout

- 请求方式 ：
    POST

- 请求实例：

        {
            "user_id":201710414227
        }
        
- 请求参数说明:        

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |user_id|对应表USERS.USER_ID的值|
  
- 返回实例：

        {         
            "status": true,
            "info": null,    

        }
 
- 返回参数说明：    
 
  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |status|true表示正确的返回，false表示有错误|
  |info|返回结果说明信息|
